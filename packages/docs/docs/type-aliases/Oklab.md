# Type Alias: Oklab

> **Oklab**: `object`

Represents a color in the Oklab color space.
'L' is the lightness component (0 to 1).
'a' is the green-red component (usually -0.4 to 0.4).
'b' is the blue-yellow component (usually -0.4 to 0.4).
