# Type Alias: HSV

> **HSV**: `object`

Represents a color in the HSV color space.
'h' (hue) should be between 0 and 360.
's' (saturation) and 'v' (value) should be between 0 and 100.
The alpha channel 'a' is optional and should be between 0 and 1 if provided.
